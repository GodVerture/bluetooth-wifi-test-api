# 无线网络与蓝牙接口管理工具

这是一个用C++编写的跨平台WiFi和蓝牙接口管理工具，提供完整的网络设备管理功能。

## 项目概述

本项目实现了以下核心功能：

### WiFi功能
- **网络扫描**：扫描可用的WiFi网络
- **网络连接**：连接指定的WiFi网络
- **网络管理**：管理已保存的网络配置
- **AP模式**：支持创建WiFi热点
- **静态IP配置**：支持设置静态IP地址
- **自动连接**：支持网络自动连接功能

### 蓝牙功能
- **设备扫描**：扫描周围的蓝牙设备
- **设备配对**：与蓝牙设备进行配对
- **设备连接**：连接已配对的蓝牙设备
- **设备管理**：管理已配对和已连接的设备
- **自动连接**：支持蓝牙设备自动连接

## 项目结构

项目采用模块化设计，主要包含以下文件：
```bash
Interface/
├── main.cpp          # 主程序入口，包含用户界面和测试代码
├── WifiInterface.h   # WiFi接口类头文件
├── WifiInterface.cpp # WiFi接口类实现
├── BlueInterface.h   # 蓝牙接口类头文件
├── BlueInterface.cpp # 蓝牙接口类实现
├── Makefile          # 构建配置文件
└── README.md         # 项目说明文档
```

## 编译与运行

### 环境要求
- C++11兼容的编译器 (g++ 4.8+ 或 clang 3.3+)
- Linux系统

### 编译命令
```bash
# 使用默认编译器
make

# 使用交叉编译工具链（适用于嵌入式设备）
make CROSS_COMPILE=aarch64-none-linux-gnu-
```

### 运行程序
```bash
./Peripheral_interface_test
```

## 使用说明

### WiFi功能测试
在`main.cpp`中，取消注释以下宏定义来启用WiFi测试：
```bash
#define WIFI_TEST
```

WiFi功能包括：
- 扫描可用网络
- 连接/断开网络
- 管理已保存网络
- 设置静态IP
- 创建WiFi热点

### 蓝牙功能测试
在`main.cpp`中，取消注释以下宏定义来启用蓝牙测试：
```bash
#define BLUE_TEST
```

蓝牙功能包括：
- 扫描蓝牙设备
- 配对/取消配对设备
- 连接/断开设备
- 管理已配对设备
- 设置自动连接

## 平台支持

- 仅支持Linux平台，Windows平台未做适配

## 注意事项

1. **权限要求**：部分功能需要管理员权限执行
2. **硬件依赖**：现接口使用的为rtl88xx系列的蓝牙WiFi芯片进行测试
3. **平台差异**：不同平台的功能实现可能有所差异
4. **测试模式**：建议在测试环境中使用，避免影响生产环境

## 开发说明

### 添加新功能
1. 在对应的接口类中添加方法声明
2. 在实现文件中添加平台特定的实现
3. 在主程序中添加相应的测试代码

### 调试
- 使用条件编译控制功能模块
- 分模块测试各个功能
